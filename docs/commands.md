# Available Commands

For reference here is a list of all available commands the editor currently understands.  
There are some useful commands missing (like Images and Links). But they are already planned.

## Strong

```
<button type="button"
    data-command="strong"
>
</button>
```

## Emphasize / Italic

```
<button type="button"
    data-command="em"
>
</button>
```

## Inline Code

```
<button type="button"
    data-command="code"
>
</button>
```

## Ordered List

```
<button type="button"
    data-command="ordered_list"
>
</button>
```
